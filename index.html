MedicHelper

Desenvolvido em Java, utilizando o NetBeans e banco de dados MySQL, o software tem como objetivo facilitar o atendimento aos pacientes. O sistema conta com autenticação por ID e senha. Os pacientes podem acessar o sistema para agendar consultas na especialidade desejada, além de terem a opção de cancelar ou alterar os horários das consultas agendadas. Os médicos possuem acesso restrito, permitindo apenas a visualização das consultas e dos horários agendados de seus pacientes. Por fim, o sistema conta com um usuário administrador (Adm), responsável por gerenciar toda a aplicação, incluindo a exclusão de registros no banco de dados.

Tela de Login

Nessa primeira interface o usuário deve selecionar o nível de acesso, conforme cadastrado pelo administrador e selecionar o usuário e senha correspondente.

Tela Administrador

Tendo acesso a essa tela, o administrador tem total controle a todo banco de dados do sistema. Podendo cadastrar pacientes, médicos e excluir consultas.

Cadastro de Médicos

Nessa interface autenticado como administrador, podemos fazer o cadastro de médicos no sistema, incluindo atualizar ou excluir médicos.

Cadastro de Pacientes

Autenticado como um administrador, tendo acesso a essa interface. Podemos cadastrar pacientes, atualizar cadastros ou excluí-los.

Pacientes

O paciente, após efetuar o login com o id e senha. Irá se deparar essa interface, onde poderá agendar a sua consulta com a especialidade que necessita. Nossa ideia era fazer o atendimento por ordem de chegada!

Consultas Agendadas

Essa interface é do médico, após efetuar o login com o id e senha. O médico só vai poder visualizar as consultas e suas datas.

Lista de Pacientes

Nessa mesma interface do médico, também poderar exibir os pacientes que estão cadastrados com a sua especialidade.
