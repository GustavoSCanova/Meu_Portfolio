MedicHelper

Desenvolvido com Java utilizando o netbeans e banco de dados MySQL. O software tem como objetivo facilitar o atendimento aos pacientes. Cujo o mesmo tem um usuário e senha para acessar o sistema. No sistema o paciente pode realizar a consulta na especialidade a qual necessita, pode cancelar ou alterar o horário. Os médicos tem o acesso somente para visualizar as consultas dos paciente e os horários agendados. Por fim temos o usuário de Adm, responsável por gerenciar todo o sistema ou deletar qualquer base de dados.
